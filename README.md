# Whatsapp API Integration 

# Requirements
<li>Node JS</li>
<li>Express JS</li>
<li>MongoDB</li>

# Features 
<li>Login</li>
<li>Get Scan Code</li>
<li>Send Message</li>

# Getting Started Locally
Make sure you have Node.js installed.
<pre>git clone https://github.com/kakadiyahitesh20/whatsapp-api.git # or clone your own fork
npm install
npm start</pre>
Your app should now be running on localhost:3000
